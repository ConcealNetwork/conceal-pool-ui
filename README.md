# Conceal Pool UI

> UI for [Conceal mining pool](https://pool.conceal.network/)

## Requirements

- Node.js 16
- Configured AWS profile (for deployment only)

## Local Development

```shell
$ git clone https://github.com/bomb-on/conceal-pool-ui.git
$ cd conceal-pool-ui
$ npm install
$ npm start
```

## Deploy

```shell
$ npm run aws:deploy
```
